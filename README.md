# SeaBattle
Známa hra "námorná bitka" kde sa snažíte zasiahnuť protivníkove lode na hracom poli. 
Hra prebieha na dvoch zariadeniach, ktoré spolu komunikujú cez bluetooth. 
Features:
  > bluetooth komunikácia
  > multimedia (zvuky)
  > 2D canvas
  > vlákna
  > persistent storage
  > ovládanie vibrácii
